# PIKA_AI

## Env
OpenAI gym from [here](https://github.com/MrHoseongLee/gym-pikachu-volleyball)

## Prerequisites
- Install [PyTorch dependencies](http://pytorch.org)
- Install packages(```poetry install```)

learning rate = 0.0002
